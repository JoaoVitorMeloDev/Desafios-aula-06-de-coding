# 📦 Sistema de Estoque (Python)

Projeto simples em Python para gerenciar produtos em um estoque.

---

## 🚀 Funcionalidades

- Adicionar produtos
- Listar produtos
- Buscar produto por nome
- Atualizar quantidade
- Remover produto

---

## 🧠 Regras

- Não permite quantidade zero ou negativa
- Não permite produtos com nomes duplicados
